# Changelog

All notable changes to this project will be documented in this file.

This project follows [Semantic Versioning](https://semver.org/).

## [0.1.0]

### Added

- Interactive terminal UI for browsing and searching internet radio stations.
- Station filtering by name, tags, country code, language, and minimum bitrate.
- Linux playback through `cvlc`.
- Windows 10/11 playback through the native Windows media backend.
- Keyboard volume controls.
- Favorites storage in the OS-native per-user config directory.
- Pagination for large station result sets.
- Cross-platform CI for Linux and Windows.
